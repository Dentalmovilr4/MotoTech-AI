# MotoTech-DMR4 AI 🏍️

Asistente de diagnóstico inteligente para motocicletas **Bajaj Pulsar NS200**, potenciado por el modelo **Gemini-3-Flash**.

## 🚀 Características
- Diagnóstico de códigos de falla (MIL).
- Valores de referencia para sensores (TPS, O2, EOT).
- Guía de mantenimiento preventivo.

## 🛠️ Configuración
1. Agrega tu `GOOGLE_API_KEY` en los Secrets de este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`.
3. Ejecuta el asistente: `python app.py`.
